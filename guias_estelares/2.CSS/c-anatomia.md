# Anatomia

```css

h1 {
    color: blue;
    font-size: 60px;
    background: gray;
}

```

* Selector (Seletores) -> h1
* Declaration (Declaração) -> todo o código dentro das chaves
* Properties (Propriedades) -> valor antes dos dois pontos (características)
* Property values (Valores) -> valores depois dos dois pontos
